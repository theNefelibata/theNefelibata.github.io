    • LEETCODE 存在重复元素

    II题目：

    给定一个整数数组和一个整数 k，判断数组中是否存在两个不同的索引 i 和 j，使得 nums [i] = nums [j]，并且 i 和 j 的差的 绝对值 至多为 k。

    示例 1:

输入: nums = [1,2,3,1], k = 3
输出: true
示例 2:

输入: nums = [1,0,1,1], k = 1
输出: true
示例 3:

输入: nums = [1,2,3,1,2,3], k = 2
输出: false

    III题解：

    维护一个大小为K的队列Q，遍历nums，如果nums[i]在队列Q中，则返回True，否则，将nums[i]加入队列Q，将Q多余元素出队。

    class Solution:
	def containsNearbyDuplicate(self, nums: List[int], k: int) -> bool:
		queue = []
		if k == 0:
			return False
		for i in nums:
			if i not in queue:
				if len(queue) < k:
					queue.append(i)
				else:
					queue.pop(0)
					queue.append(i)
			else:
				return True
		return False

    III题目：

    给你一个整数数组 nums 和两个整数 k 和 t 。请你判断是否存在 两个不同下标 i 和 j，使得 abs(nums[i] - nums[j]) <= t ，同时又满足 abs(i - j) <= k 。

    如果存在则返回 true，不存在返回 false。

    示例 1：

输入：nums = [1,2,3,1], k = 3, t = 0
输出：true
示例 2：

输入：nums = [1,0,1,1], k = 1, t = 2
输出：true
示例 3：

输入：nums = [1,5,9,1,5,9], k = 2, t = 3
输出：false

    III题解

    从头开始遍历数组，对于当前下标i，为下标为[i-k,i+k]的元素构造桶🪣，对于每个i我们只需看前面是否有无效桶，将nums[i-(k+1)]的桶删除。

    对于元素nums[i]，获取当前桶id，如果存在的话说明找到符合条件的元素，如果当前桶没有元素，则比较相邻桶，有元素的话做差比较，如果没找到，将当前元素装入桶中。

    • 分桶时，如果t=0, 则nums[i]//t 不可行，考虑nums[i]//t+1 ;
    • 若nums[i]<0，进行(x+1)//(t+1)-1

    III代码

    class Solution:
	def containsNearbyAlmostDuplicate(self, nums: List[int], k: int, t: int) -> bool:
		def getID(x, t):
			return (x+1)//(t+1)-1 if x<0 else x//(t+1)
		
		n = len(nums)
		m = {}
		for i in range(n):
			x = nums[i]
			idx = getID(x, t)
			if idx in m:
				return True
			l , r = idx-1, idx+1
			if l in m and abs(x - m[l]) <= t:
				return True
			if r in m and abs(x - m[r]) <= t:
				return True
			m[idx] = x
			if i >= k:
				m.pop(getID(nums[i-k],t))
		return False
